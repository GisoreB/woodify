# Woodify

Woodify is a Flutter-based furniture e-commerce app designed to make buying furniture online simple, fast, and enjoyable. Whether you’re furnishing your home or setting up an office, Woodify offers a wide range of high-quality furniture to match every style and budget.

With a modern user interface and smooth performance, Woodify allows users to browse collections, explore detailed product pages, search for items, and shop with ease. It’s the perfect solution for anyone looking to build or launch a professional furniture shopping app using Flutter.

## ✨ Woodify App Features

* Beautiful and modern UI built with Flutter
* Onboarding screens for first-time users
* Secure authentication (Sign up & Login)
* Browse furniture by categories and collections
* Detailed product pages with images, prices, and descriptions
* Powerful search to quickly find products
* Favorites / wishlist to save liked items
* Shopping cart for easy order management
* User profile to manage personal information
* Light and dark mode support
* Smooth animations and fast performance
* Clean, well-structured, and easy-to-customize codebase

Woodify is designed to deliver a smooth shopping experience while giving developers a strong, scalable foundation for a furniture e-commerce app.
